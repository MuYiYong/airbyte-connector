    最近邻查询¶

    最近邻查询语句用于在图中搜索向量的最近邻。该语句支持两种搜索类型：K 最近邻（k-nearest neighbor，简称 KNN）搜索和近似最近邻（Approximate Nearest Neighbor，简称 ANN）搜索。

    KNN 是一种精确搜索，返回查询向量的 K 个最近邻。它不依赖于向量索引，适用于小型的图和低维度的向量。

    ANN 是一种近似搜索，返回查询向量的近似最近邻。它依赖于向量索引，适用于大型的图和高维度的向量。

    在执行 ANN 搜索前，需要在待搜索的向量上预先创建向量索引；否则，ANN 搜索将自动转换为 KNN 搜索。更多信息，请参见 CREATE VECTOR INDEX。

    语法¶

    knn_search_statement
    ann_search_statement
    ann_search_option

    knn_search_statement ::= order_by_clause limit_clause 

ann_search_statement ::= order_by_clause ('APPROX'| 'APPROXIMATE') limit_clause ('OPTIONS' '{' ann_search_option (',' ann_search_option)* '}' )

ann_search_option::=  
    'METRIC' ':' ( 'L2' | 'IP' )
  | 'TYPE' ':' ( 'IVF' | 'HNSW' )
  | 'NPROBE' ':' int_value
  | 'EFSEARCH' ':' int_value

    参数¶

    order_by_clause¶

    基于查询向量和索引向量之间的相似度对搜索结果进行排序的子句。该子句必须包含一个用于测量向量相似度的函数。

    支持以下函数：

    • euclidean()：若在 ANN 搜索中使用，需满足以下条件：
      • 已为待搜索的向量创建度量为L2的向量索引；否则，ANN 搜索会自动转换为 KNN 搜索。
      • 搜索结果按升序排序。
    • inner_product()：若在 ANN 搜索中使用，需满足以下条件：
      • 已为待搜索的向量创建度量为IP的向量索引；否则，ANN 搜索会自动转换为 KNN 搜索。
      • 搜索结果按降序排序。
    • cosine()：若使用该函数，则仅支持 KNN 搜索。

    更多信息，请参见 ORDER BY。

    limit_clause¶

    限定返回的搜索结果数量的子句。更多信息，请参见 LIMIT。

    ann_search_option¶

    执行 ANN 搜索的选项。

    METRIC¶

    用于衡量查询向量和索引向量之间相似度的度量。该值会根据前面order_by_clause中指定的函数自动设置，规则如下：

    • 若函数为euclidean()，则度量默认设置为L2，即根据向量之间的欧几里得距离计算相似度。更多信息，请参见 euclidean()。
    • 若函数为inner_product()，则度量默认设置为IP，即根据向量之间的内积计算相似度。更多信息，请参见 inner_product()。

    若此处指定的值与前面order_by_clause中指定的函数相矛盾，则会出现报错提醒。

    TYPE¶

    用于执行 ANN 搜索的向量索引类型。有效值为IVF和HNSW。若未指定此选项，则默认值IVF生效。更多信息，请参见 CREATE VECTOR INDEX。

    NPROBE¶

    在 IVF 搜索中搜索的聚类数量。该选项仅在选项TYPE设置为IVF时生效。

    该选项用于权衡 IVF 算法的搜索速度和搜索准确度。取值越大，搜索准确度越高，但搜索速度越慢。

    有效值范围为1到NLIST，其中NLIST为创建 IVF 索引时创建的聚类数量。更多信息，请参见 NLIST。

    若NPROBE设置为与NLIST相同的取值，则理论上搜索的召回率最高，但搜索速度最慢。

    若未指定该选项且TYPE设置为IVF，则默认值8生效。

    EFSEARCH¶

    在 HNSW 搜索中动态候选列表的大小，该列表大小决定了搜索过程中评估的邻居数量。该选项仅在选项TYPE设置为HNSW时生效。

    该选项用于权衡 HNSW 算法的搜索速度和搜索准确度。取值越大，搜索准确度越高，但搜索速度越慢。

    有效值范围为1到无穷大。

    若未指定该选项且TYPE设置为HNSW，则默认值16生效。
