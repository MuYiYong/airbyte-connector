    Storage 服务配置项¶

    本文列出仅作用于 Storage 服务的服务级配置项。关于如何为 Storage 服务设置服务级配置项，请参见设置方式。

    表头描述¶

    下文表格中表头说明如下：

    字段 说明
    名称 配置项的唯一标识符。
    数据类型 配置项值的数据类型。
    默认值 系统预定义的配置值（未主动设置时的默认值）。
    预设值 配置文件中的初始值（若存在）。优先级：预设值 > 默认值。更多信息，请参见优先级规则。
    描述 功能说明、作用范围及注意事项。
    引入版本 首次支持的该配置项的悦数图数据库版本号。
    取值范围 合法值的定义规则或区间。
    动态生效 是否支持运行时动态修改（无需重启服务）。

    minimum_reserve_space_mb Int 128 - 定义磁盘空间预留阈值。根据可用磁盘空间调节写入来防止磁盘写满。
